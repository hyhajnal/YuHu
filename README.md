## 语乎Hybrid版
一个小学生词语听写的Hybird App，分为教师端和学生端。

### 功能
* 登录注册
* 语音听写
* canvas手写面板

### 结构
* 客户端 Appcan + jQuery
* 后端 Express + Mysql
* [apk](./demo/语乎.apk) (只打包了android)
